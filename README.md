# Employee Management System

## Project Description

This project is a simple Employee Management System developed using **PHP, MySQL, and Bootstrap**.
The system allows users to manage employee information including adding, updating, and deleting records.

## Features

* Insert new employee data
* Update existing employee information
* Delete employee records
* Display employee list in a table
* Auto-generate Employee ID (EMP001, EMP002, etc.)

## Technologies Used

* PHP
* MySQL
* HTML / CSS
* Bootstrap 5
* JavaScript

## Database

Database name: **sales_order_db**

Table used in this program:

* `employee`

Fields:

* `emp_id`
* `emp_name`
* `age`
* `phone`

## How to Run the Project

1. Import the database file `sales_order_db.sql` into MySQL.
2. Place the project folder inside the **htdocs** directory (for XAMPP).
3. Start **Apache** and **MySQL** in XAMPP.
4. Open a browser and go to:

```
http://localhost/project-folder-name/index.php
```
